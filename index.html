Path Tracing made

 

strahl is an open-source path tracing library built using WebGPU.

Get StartedGithub

Modern Technology

Strahl leverages WebGPU, the latest web graphics API built for years to come.

Fully Open-Source

Strahl is built out in public under MIT license, your feedback is welcome.

Using OpenPBR

Strahl supports Physically-based Rendering based on the OpenPBR surface shading model.
